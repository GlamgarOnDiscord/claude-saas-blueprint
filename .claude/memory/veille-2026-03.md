## Optimisation Coûts
- Prompt caching = **-90% coûts, -85% latence**
- Sans caching : session Opus = $50-100, avec = $10-19
- `/batch` = 50% moins cher
- Moyenne : $6/dev/jour, $100-200/mois
- Thinking tokens = output tokens (facturés)
- **Piège** : timestamp dans system prompt invalide le cache
