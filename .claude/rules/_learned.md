# Rules Apprises

## Concept
Ce fichier contient les règles que l'agent apprend au fil du temps en faisant des erreurs ou en découvrant des spécificités du projet.

## Format
```
- [YYYY-MM-DD] NE PAS / TOUJOURS + action + raison
```

## Règles
<!-- Ajoutées automatiquement par l'agent quand il fait une erreur -->
<!-- Exemple : -->
<!-- - [2026-03-17] NE PAS recréer middleware.ts — le projet utilise un fichier proxy.ts à la place -->
<!-- - [2026-03-17] TOUJOURS utiliser bun au lieu de npm — c'est le package manager du projet -->
